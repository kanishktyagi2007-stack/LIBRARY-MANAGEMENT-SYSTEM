# 📚 Library Management System

## 📌 Description
This is a Python-based Library Management System that allows users to manage books efficiently. It supports issuing and returning books with fine calculation.

## ⚙️ Features
- View available books
- Issue book with student name and date
- Return book with fine calculation
- Dictionary-based data storage
- Menu-driven interface

## 💰 Fine Policy
- 1st week: ₹10/day/book
- 2nd week: ₹20/day/book
- 3rd week: ₹60/day/book
- Fine increases weekly

## 🛠️ Technologies Used
- Python
- datetime module

python main.py
